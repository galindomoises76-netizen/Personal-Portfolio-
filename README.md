## ğŸ“ Customization

### Update Personal Information

1. **Hero Section**: Edit the hero title, subtitle, and description in `index.html`
2. **About Section**: Update the about text and statistics
3. **Skills**: Modify the skill categories and tags
4. **Projects**: Replace placeholder projects with your actual projects
5. **Contact**: Update email, LinkedIn, and GitHub links

### Change Colors

Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --primary-dark: #4f46e5;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

### Add Projects

Add new project cards in the projects section of `index.html`:
```html
<div class="project-card">
    <div class="project-image">
        <!-- Project image or placeholder -->
    </div>
    <div class="project-content">
        <!-- Project details -->
    </div>
</div>
```
